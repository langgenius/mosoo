简历优化器

Resume Optimizer
粘贴你的简历，由 Claude 给出评分、可执行的修改建议，并重写一版可直接使用的简历。

它会帮你做什么

  • 综合评分：从 0–100 评估简历当前质量。
  • 优势 / 短板：指出真正影响录用的问题，而非泛泛而谈。
  • 逐条改写建议：按影响力排序，给出可直接套用的措辞。
  • ATS 关键词：对照 JD，标出已命中与缺失的关键词。
  • 重写版简历：应用全部建议后的 Markdown 成稿，可一键复制。

模型不会编造你未提供的数字 —— 缺失指标会以 [X%] 占位，方便你自行填写。
